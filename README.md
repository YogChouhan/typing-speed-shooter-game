# Typing Speed Shooter Game 🎮⌨️

A fun and interactive browser-based game that combines typing practice with shooter-style gameplay. Improve your typing speed, accuracy, and reflexes while destroying enemies by typing the correct words before they reach you.

---

## 📌 Features

- 🎯 Real-time typing gameplay
- ⚡ Improve typing speed and accuracy
- 💥 Shooter-style mechanics
- 🧠 Dynamic word spawning
- 📈 Score tracking system
- ⌨️ Fast-paced and engaging gameplay
- 🎨 Interactive UI and animations
- 📱 Responsive design support

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 📂 Project Structure
The repository consists of the following core files:

- `entry.html` - The landing/start screen of the game.
- `main.html` - The core gameplay interface where the action happens.
- `gg.html` - The "Game Over" screen displaying final stats and WPM.
- `script.js` - Contains all the core mechanics and JavaScript logic.
- `style.css` - The stylesheet for making the game look great.
- `logo.png` - Logo asset for the game.
---

## 🎮 How to Play

1. Start the game
2. Words/enemies will appear on the screen
3. Type the displayed word correctly
4. Correct typing destroys the enemy and increases your score
5. Survive as long as possible and achieve the highest score

---

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/YogChouhan/typing-speed-shooter-game.git
```

Open the project folder:

```bash
cd typing-speed-shooter-game
```

Run the game by opening:

```bash
index.html
```
in your browser.
